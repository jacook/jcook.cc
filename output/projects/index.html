My Projects

  • The CitizenWeb Project
    The CitizenWeb Project is a mission to fight for a free, open, and above all a decentralized Internet. In order to achieve this, it aims to empower everyday internet users with the information and resources they need to take matters into their own hands. It provides news, features, guides and software to these ends. I am the Project's founder and editor.
  • 1984
    1984 is a planned cross-platform social networking client for the Tent protocol. It will feature short status posts, longer essay posts, multiple account support, and (eventually) message encryption between small groups. This project is being produced in conjunction with The CitizenWeb Project.
  • ARK
    ARK is a planned set of guides and startup scripts for easily running server software on the Raspberry Pi. The goal is to make it easier and less expensive for individuals to self-host their own data and assure privacy. This project is being produced in conjunction with The CitizenWeb Project.
  • KnockKnock
    KnockKnock is a simple and clean yet robust blog system for use with the nanoc static site generator. This site is built on it!
